# Security Policy

## Reporting a Vulnerability
If you discover a security issue, please do not open a public issue immediately.

Instead, report it privately with:
- Affected file/component
- Reproduction steps
- Potential impact
- Suggested mitigation (if known)

## Response Targets
- Initial acknowledgement: within 72 hours
- Triage decision: within 7 days
- Patch timeline: depends on severity and complexity

## Disclosure
Once fixed, we may publish a summary including impact and remediation guidance.
